⛺️ 2022 HSU Coding Camp - JavaLadderGame

## 설명
Java GUI 기반 사다리게임

## 관련 이미지
### 초기 화면
우측 **"시작"** 버튼을 눌러 게임을 시작한다.<br>
게임 player는 상단 Label을 통해 사용자가 입력할 수 있다.<br>
